# Scratch4D
Scratch for drones.

###Setup and Running
```npm install```
```npm start```

###Supported drone platforms
A.R. Drone 2.0
Bebop Drone(working)

###Supported Environment
Windows 7/8.1/10  
Mac(No support for video streaming)

###Requirements
Electron
Node.js
Drone  


###Packaging
```asar pack . Scratch4D.asar```
