# Scratch4D
Scratch for drones.

###Setup and Running
```npm install```
```npm run start```

###Supported drone platforms
A.R. Drone 2.0
Bebop Drone(working)

###Supported Environment
Windows 7/8.1/10  
Mac(No support for video streaming)

###Requirements
Electron
Node.js
Drone  


###Building
1. clean files  
```
npm run clean
```    
2. building files
```
npm run build
```
